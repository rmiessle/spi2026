# Gettysburg College Summer Pedagogy Institute 2026
## Wrangling Humanities Data with AI and R: A Very Brief Introduction
### Facilitated by R.C. Miessler, Digital Initiatives Librarian
Yes, humanists use data too. This session will provide a very brief introduction to using AI tools to write R scripts to complete a basic analysis of structured and unstructured data. And if none of that sentence makes sense, this may be the session for you. Please bring your laptop. No coding experience is necessary!
